## Getting Started

**1. Create a virtual environment** 

```bash
python3 -m venv env
```

**2. Activate the virtual environment**
On macOS/Linux:

```bash
source env/bin/activate
```

**3. Install dependencies package with**`requirements.txt`

```bash
pip3 install -r requirements.txt
```

**4. Run the server**

```bash
python3 server.py
```
